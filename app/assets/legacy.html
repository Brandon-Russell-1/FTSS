FTSS 3 Legacy Migration Page


It appears that you are using an older version of Internet Explorer (IE) or have forced "compatiblity mode" on for your web browser.
FTSS 3 requires IE 10, an AF-approved modern web browser that you can load right now without administrator rights.


To update your browser, please follow these steps:

  1. Open Control Panel
  2. Search for "Advertised Programs"
  3. Open "Run Advertised Programs"
  4. Choose the Internet Explorer 10 Update
  5. Follow the installation instructions
