FTSS 3 Development Progress

This page lists UI/design changes and development progress for FTSS 3.

27 Jan - Schedule, Requests, Catalog, Units Video Demo

This video covers the basic search and view features as well as some of the page-to-page navigation and data grouping. Currently working on updating some of the color schemes for readablity & completing the missing processes.

Seat Request Form

The seat request is opened as a modal dialog when a UTM or MTF clicks "Request Seats". Students are added from the students list which is linked to the AF GAL so rank/unit should always be up-to-date. The form will only allow the requester to fill available seats and includes the ability to leave comments related to the seat request for the FTD.

Catalog & Units (Table Views)

The table views show more detail and activate action buttons when you hover over the row. These views will include grouping when it makes sense as well as column sorting (by clicking on the header).
